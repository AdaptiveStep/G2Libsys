# G2Libsys
G2Systems - Library System Agile Implementation

## Requirement for running: 
SQL Server
.Net Core 3

## Installation: 
* 1: Clone the project
* 2: setup one connection in App.config
* 3: Run the G2Libsys.SQL in your database
* -> Press Play in visual studio.
